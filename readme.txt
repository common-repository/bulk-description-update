== Description ==

If you find that updating the description or slug for category, tag or link category one by one is annoying, this is absolutely a plug-in for you. With this plug-in, you are allowed to update slug and description for multiple records in one page. Also, it allows you to use html tags inside the description.

== Installation ==

1.Download the plugin and expand it if necessary.

2.Copy the folder named 'bulk-description-update' to the plugin folder(wp-content/plugins/)

3.Login into the WordPress administration area and go to the plugin page.

4.Click the activate link of the 'bulk-description-update' plugin.

5.Now, you can go to Posts -> Bulk Description Update and update the slug or description you want.

6.This plugin also remove the html tag filter, so you can now use html tag to embellish your display of description.


== Frequently Asked Questions ==

= 1.How do I use this plugin? = 

1.Make sure you have activated the plugin at the plugin panel.

2.Go to Posts -> Bulk Description Update

3.Select category, tag, link category, whatever you like to update, at the dropdown box.

4.All the records under the category you selected will be pulled over.

5.Update the description or slug and press update button.
